Workshops & Useful Resources

These carefully selected resources will help you learn each major step by step.

Artificial Intelligence

Start learning AI fundamentals through structured, beginner-friendly resources.

  • AI Engineer Roadmap (Roadmap.sh)
  • Machine Learning by Andrew Ng (Coursera)
  • AI Courses on EdX
  • MIT Introduction to Deep Learning
  • AI Tutorials (YouTube Playlist)

These resources help you build strong foundations before choosing advanced topics.

Cybersecurity

Begin your cybersecurity journey through hands-on learning platforms.

  • TryHackMe – Complete Beginner Path
  • HackTheBox Academy – Free Modules
  • Cybersecurity Roadmap (Roadmap.sh)
  • Networking for Beginners (YouTube)
  • IT Security – Coursera

These platforms give you real practical challenges to build confidence.

Software Engineering

Learn how software is planned, designed, and built from scratch.

  • FreeCodeCamp – Full Free Curriculum
  • W3Schools – HTML, CSS, JS, Python
  • Full Stack Roadmap (Roadmap.sh)
  • Harvard CS50 – Free Computer Science Course
  • Traversy Media (YouTube) – Coding Tutorials

Start with basic programming, then move into project building.

Game Development

These resources help you create your first games with real tools.

  • Unity Learn – Free Courses
  • Unreal Engine Beginner Tutorials
  • Game Developer Roadmap (Roadmap.sh)
  • Free Game Assets (itch.io)
  • Unity 2D Game Tutorial (YouTube)

Game development combines creativity, art, and programming into one pathway.
